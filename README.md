# 🗨️ One-One and Group Chatting App

A real-time messaging application that supports both **one-to-one** and **group chats**, built using Java Swing, AWT, Socket Programming, and .NET. The app features a responsive UI and a robust client-server architecture for efficient communication.

---

## 🚀 Features

- 💬 **Private Chatting** – One-to-one real-time conversations between users.
- 👥 **Group Chatting** – Allows multiple users to communicate in a shared chat room.
- 🧑‍💻 **Cross-platform Compatibility** – Java Swing/AWT for UI and .NET for backend integration.
- 🔐 **Stable Connections** – Built using socket programming for real-time, low-latency communication.
- 🎨 **User-Friendly Interface** – Designed with Java AWT and Swing for smooth user interactions.

---

## 🛠️ Technologies Used

| Layer          | Technology Used      |
|----------------|----------------------|
| Frontend (UI)  | Java Swing, AWT       |
| Networking     | Java Socket Programming |
| Backend        | .NET Framework         |
